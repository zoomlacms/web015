中南集团组织学习十八届三中全会会议精神
访问数：221 最后更新：2013年11月19日 11时03分 字体大小：[大][中][小] 打印

十八届三中全会召开后，全国上下掀起了学习十八届三中全会精神的热潮，中南集团也不例外。11月18日下午，在总部会议中心，控股集团董事长陈锦石带领各产业集团一级部门以上负责人共同学习了十八届三中全会会议精神。

　　

十八届三中全会是在中国改革开放事业进入到关键时刻所召开的会议，会议在政府职能转变、城镇化、金融开放、财政税收、资源定价等领域作出重要决策，就中国全面深化改革进行总体部署。英国《金融时报》报道说，十八届三中全会将成为中国经济、社会与政治全面发展的里程碑。

　　

会上，董事长陈锦石从政治体制改革、司法独立、市场经济的角度全面剖析了十八届三中全会的会议精神，尤其对《深化改革决定》中关于市场经济的部分表示高度认同，他认为60条改革意见是中国政治经济发展进步的表现，要抓住这样一个机遇，在改革的浪潮中寻求发展。他对在座的管理人员提出这样的要求，对十八届三中全会会议精神要有高度的认识，一只脚踩油门，一只脚踩刹车的时代已经过去，四大产业集团已进入攻坚阶段，要对公司未来的发展做全面研究和规划部署，并最终实现落地。

　　

此外，董事长还下一步改革整顿工作也做了进一步要求。他指出，全年的工作已经进入到了关键时刻，要将整顿工作抓得有声有色，要触动各项工作推进，触动各项经济指标的完成，努力实现全集团整体发展。
